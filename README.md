# Code blocks (WIP)

Custom React component to display code blocks and corresponding notes, ideal for lecture notes!

## Tech stack

- React + Typescript
- Vite
- Prettier + ESLint (formatting)

## Other tools used

- [Shiki syntax highlighter](https://shiki.style/)
